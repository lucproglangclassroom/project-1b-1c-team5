# LLM Usage

I used ChatGPT as a support tool during development of the `topwords` project.

## Areas where it helped
- clarifying the project structure
- suggesting test cases for the sliding-window behavior
- helping troubleshoot build and coverage setup
- reviewing command-line and logging setup

## Files affected
- `src/test/scala/impl/TopWordsProcessorTest.scala`
- `src/test/scala/impl/TopWordsMainTest.scala`
- `src/test/scala/impl/ConsoleObserverTest.scala`
- `build.sbt`

## Review process
All suggestions were reviewed, edited, and tested before being included in the project.
